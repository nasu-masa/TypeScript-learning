# TypeScript 学習リポジトリ

## リポジトリ概要

TypeScript の学習記録・復習用リポジトリです。
教材の内容をそのまま写すのではなく、自分の言葉でコードや説明を追記していくための土台として使用します。

## 学習目的

- TypeScript の基礎から応用までを体系的に学ぶ
- 自分の理解をコードとメモで記録・整理する

## Chapter 一覧

| Chapter | テーマ |
|---------|--------|
| [Chapter 01 - Basics](./chapter01-basics/) | 型の基礎 |
| [Chapter 02 - Environment](./chapter02-environment/) | 開発環境 |
| [Chapter 03 - Advanced](./chapter03-advanced/) | 応用 |
| [Chapter 04 - Best Practices](./chapter04-best-practices/) | ベストプラクティス |

## 使用方法

```bash
# 依存関係のインストール
npm install

# TypeScript のコンパイル
npm run build

# ts-node で直接実行
npx ts-node <ファイル名>
```
