# Chapter 01 - Basics

## 気づき

- JavaScriptと違い型を指定できる
- 型推論があるので毎回書く必要はない
- 配列の型定義には2つの書き方があるが number[] = [1, 2, 3, 4, 5];の書き方のがよく使われている。
- Union型...複数の型のどれかが入る
- interface：オブジェクトの型に名前を付ける
- type：型に名前を付ける（オブジェクト以外にも使える）
- オブジェクトの型定義は基本的にinterfaceを使うことが多い
- readonly は、指定したプロパティへの再代入を禁止する。
- ただし、そのプロパティの中にあるオブジェクトの中身までは自動では readonly にならない。
- 中身のプロパティまで変更不可にしたい場合は、内側のプロパティにも readonly を付ける。
